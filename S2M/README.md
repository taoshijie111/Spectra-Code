## Archived S2M Source

This directory is retained only as archived source and configuration material from an earlier project state.

It is not the primary reviewer-facing entry point for this repository.

For peer review, use the repository root:

- `README.md`
- `data/`
- `scripts/`
- `examples/`
- `docs/`

Historical notebooks, generated figures, and training logs were intentionally removed from the public repository to reduce ambiguity about the maintained reviewer-facing workflow.
